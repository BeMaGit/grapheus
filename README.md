# Grapheús

Grapheús ist eine eigenständige HTML-Anwendung für ablenkungsfreies Schreibtraining mit biblischen Texten (Einheitsübersetzung).

## Installation

Es ist keine Installation erforderlich. Die Anwendung besteht aus einer einzelnen HTML-Datei.

1.  Laden Sie die Datei `grapheus.html` herunter.
2.  Speichern Sie sie an einem beliebigen Ort auf Ihrem Computer.

## Starten der Anwendung

1.  Navigieren Sie zu dem Ordner, in dem Sie `grapheus.html` gespeichert haben.
2.  Öffnen Sie die Datei mit einem modernen Webbrowser (z.B. Chrome, Firefox, Safari oder Edge).

## Benutzung

Nach dem Öffnen der Anwendung sehen Sie die Benutzeroberfläche von Grapheús.

1.  **Beginnen**: Drücken Sie eine beliebige Taste, um den Fokus auf das Eingabefeld zu legen.
2.  **Schreiben**: Tippen Sie den angezeigten Text ab.
    *   Falsche Eingaben werden rot markiert.
    *   Richtige Eingaben lassen den Cursor vorrücken.
3.  **Statistik**: Oben rechts sehen Sie Ihre aktuelle Schreibgeschwindigkeit (WPM - Wörter pro Minute) und Genauigkeit (Accuracy).
4.  **Navigation**: (Falls verfügbar) Nutzen Sie die Dropdown-Menüs oben links, um Bücher und Kapitel auszuwählen.
5.  **Themes**: Wählen Sie oben rechts ein Farbschema aus:
    *   **Dark**: Dunkler Hintergrund, ideal für wenig Licht (Standard).
    *   **Light**: Heller Hintergrund mit dunklem Text.
    *   **Sepia**: Ein augenschonender, warmer Farbton.

## Funktionen

*   **Offline-fähig**: Funktioniert ohne Internetverbindung.
*   **Keine Datenübertragung**: Alle Eingaben bleiben lokal in Ihrem Browser.
*   **Sofortiges Feedback**: Visuelle Rückmeldung beim Tippen.

## Quelle

Die Texte basieren auf der Einheitsübersetzung: [https://bibel.github.io/EUe/](https://bibel.github.io/EUe/)
